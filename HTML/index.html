Central Repository

The project consists of a central repository containing various blood deposits available

Blood Deposits Details

These details include blood type, storage area and date of storage. These details help in maintaining and monitoring the blood deposits.

Check For needed Blood Deposits

The project is an online system that allows to check whether required blood deposits of a particular group are available in the blood bank.

More Features

the system also has added features such as patient name and contacts, blood booking and even need for certain blood group is posted on the website to find available donors for a blood emergency.
